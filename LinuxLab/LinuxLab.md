Ejercicio 1 : 

Para crear el directorio con sus correspondinetes subdirectorios y archivos, utilizamos el siguiente comando :

```
> mkdir -p foo/{dummy,empty}
```

para crear el archivo se utliliza el comando touch del siguiente modo: 

```
> touch file{1..2}.txt
```
